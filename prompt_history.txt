## 주요 기능

1. 데이터 분석 (analyze)
   - SMS 메시지 길이, 단어 수, 레이블 분포 등 분석
   - 시각화 결과 생성

2. 모델 학습 (train)
   - 텍스트 전처리 및 토큰화
   - TF-IDF 벡터화 및 랜덤 포레스트 분류기 학습
   - 모델 평가 및 시각화

3. 예측 (predict)
   - 학습된 모델을 사용하여 새로운 메시지 분류
   - 예측 결과 저장

4. 모델 평가 (evaluate)
   - 정확도, 정밀도, 재현율, F1 점수 등 평가 지표 계산
   - 혼동 행렬, ROC 곡선, 정밀도-재현율 곡선 시각화

## 개선 사항

1. konlpy 패키지 없이도 기본 토큰화 방식으로 동작
2. scikit-learn 패키지 없이도 기본 기능 사용 가능
3. 한글 메시지 처리 지원
4. 다양한 시각화 결과 제공
